# Tech Stack

Casa de Historias is built with modern Android development tools and libraries.

## Core Technologies

### Kotlin

The app is written entirely in **Kotlin** with the following configurations:

```kotlin
kotlinOptions {
    jvmTarget = "17"
}
```

- **Java Version**: 17
- **Min SDK**: 26 (Android 8.0)
- **Target SDK**: 35 (Android 15)
- **Compile SDK**: 35

See `app/build.gradle.kts:14` for build configuration.

## UI Framework

### Jetpack Compose

Modern declarative UI toolkit for Android:

```kotlin
dependencies {
    implementation(platform(libs.androidx.compose.bom))
    implementation(libs.androidx.compose.ui)
    implementation(libs.androidx.compose.ui.graphics)
    implementation(libs.androidx.compose.ui.tooling.preview)
    implementation(libs.androidx.compose.material3)
    implementation(libs.androidx.compose.material.icons.extended)
    implementation(libs.androidx.navigation.compose)
}
```

**Key Libraries:**
- **Material 3** - Modern Material Design components
- **Navigation Compose** - Type-safe navigation
- **Extended Icons** - Additional Material Design icons

See `app/build.gradle.kts:50` for Compose dependencies.

## Backend Services

### Firebase

Complete backend solution with multiple services:

```kotlin
// Firebase
implementation(platform(libs.firebase.bom))
implementation(libs.firebase.analytics)
implementation(libs.firebase.firestore)
implementation(libs.firebase.auth)
implementation(libs.firebase.storage)
```

**Firebase Services:**
- **Firebase Authentication** - User authentication with email/password
- **Cloud Firestore** - NoSQL cloud database for stories and user data
- **Firebase Storage** - Cloud storage for images and audio files
- **Firebase Analytics** - Usage analytics and crash reporting

See `app/build.gradle.kts:59` for Firebase dependencies.

## Database

### Room

Local SQLite database for offline caching:

```kotlin
// Room
implementation(libs.androidx.room.runtime)
implementation(libs.androidx.room.ktx)
ksp(libs.androidx.room.compiler)
```

**Features:**
- Offline story caching
- Favorites management
- Type-safe SQL queries
- Kotlin Coroutines support

See `app/build.gradle.kts:78` for Room dependencies.

## Mapping

### MapLibre

Open-source mapping library (Google Maps alternative):

```kotlin
// MapLibre (replaces Google Maps)
implementation(libs.maplibre.android)
```

**Why MapLibre?**
- Open-source and free
- No API keys required
- Customizable map styles
- Suitable for community-focused apps

See `app/build.gradle.kts:66` for MapLibre integration.

## Dependency Injection

### Hilt

Dagger-based dependency injection:

```kotlin
// Hilt
implementation(libs.hilt.android)
ksp(libs.hilt.compiler)
implementation(libs.hilt.navigation.compose)
```

```kotlin
plugins {
    alias(libs.plugins.hilt)
    alias(libs.plugins.ksp)
}
```

**Features:**
- Compile-time validation
- ViewModel injection
- Scoped dependencies
- Easy testing

See `app/build.gradle.kts:83` for Hilt dependencies and `app/build.gradle.kts:6` for plugins.
