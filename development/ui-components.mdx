# UI Components

Casa de Historias uses Jetpack Compose for a fully declarative UI with Material 3 design.

## Main Container

The app's root composable manages navigation and bottom navigation bar.

```kotlin
@Composable
fun MainContainer() {
    val navController = rememberNavController()
    val authViewModel: AuthViewModel = hiltViewModel()

    // Determine start destination based on auth state
    val startDestination = if (authViewModel.isLoggedIn) {
        Screen.Discover.route
    } else {
        Screen.Welcome.route
    }

    Scaffold(
        containerColor = NocheProfunda,
        bottomBar = {
            if (showBottomBar) {
                NavigationBar { /* ... */ }
            }
        }
    ) { padding ->
        NavGraph(
            navController = navController,
            authViewModel = authViewModel,
            startDestination = startDestination
        )
    }
}
```

### Bottom Navigation Behavior

The bottom bar only shows on main screens:

```kotlin
val showBottomBar = currentRoute in listOf(
    Screen.Discover.route,
    Screen.StoryList.route,
    Screen.StoryMap.route,
    Screen.Favorites.route,
    Screen.Profile.route
)
```

Detail screens, auth screens, and secondary screens hide the bottom bar.

## Key Screens

### StoryListScreen

Displays all stories in a scrollable list with modern cards:

```kotlin
@Composable
fun StoryListScreen(
    viewModel: StoryViewModel,
    onStoryClick: (String) -> Unit = {},
    onAddStoryClick: () -> Unit = {},
    onSearchClick: () -> Unit = {}
) {
    val stories by viewModel.stories.collectAsState()
    val isLoading by viewModel.isLoading.collectAsState()
    val errorMessage by viewModel.errorMessage.collectAsState()

    Scaffold(
        topBar = { /* Top app bar */ },
        floatingActionButton = {
            FloatingActionButton(
                onClick = onAddStoryClick,
                containerColor = AguaSagrada,
                contentColor = NocheProfunda
            ) {
                Icon(Icons.Default.Add, contentDescription = "Añadir Historia")
            }
        }
    ) { padding ->
        LazyColumn {
            items(stories, key = { it.id }) { story ->
                ModernStoryCard(
                    story = story,
                    onClick = { onStoryClick(story.id) }
                )
            }
        }
    }
}
```

### ModernStoryCard

A card component displaying story information with image background:

```kotlin
@Composable
fun ModernStoryCard(
    story: Story,
    onClick: () -> Unit = {}
) {
    Card(
        modifier = Modifier
            .fillMaxWidth()
            .height(200.dp),
        shape = RoundedCornerShape(24.dp),
        onClick = onClick
    ) {
        Box(modifier = Modifier.fillMaxSize()) {
            // Background image if available
            if (!story.imageUrl.isNullOrBlank()) {
                AsyncImage(
                    model = story.imageUrl,
                    contentDescription = null,
                    modifier = Modifier.fillMaxSize(),
                    contentScale = ContentScale.Crop
                )
                // Gradient overlay for readability
                Box(
                    modifier = Modifier
                        .fillMaxSize()
                        .background(
                            Brush.verticalGradient(
                                colors = listOf(
                                    Color.Transparent,
                                    NocheProfunda.copy(alpha = 0.6f),
                                    NocheProfunda.copy(alpha = 0.95f)
                                )
                            )
                        )
                )
            }

            Column(
                modifier = Modifier
                    .fillMaxSize()
                    .padding(20.dp),
                verticalArrangement = Arrangement.Bottom
            ) {
                // Nahuatl title
                if (story.titleNahuatl.isNotBlank()) {
                    Text(
                        text = story.titleNahuatl,
                        fontStyle = FontStyle.Italic,
                        color = AguaSagrada
                    )
                }
                // Spanish title
                Text(
                    text = story.titleEs,
                    style = MaterialTheme.typography.titleLarge,
                    fontWeight = FontWeight.Bold,
                    color = LunaLlena
                )
                // Narrator and community
                Row {
                    Text(story.narratorName)
                    Text("•")
                    Text(story.community)
                }
            }
        }
    }
}
```

See `ui/story/StoryListScreen.kt:194` for the card implementation.

## State Management

Screens collect state from ViewModels using `collectAsState()`:

```kotlin
val stories by viewModel.stories.collectAsState()
val isLoading by viewModel.isLoading.collectAsState()
val errorMessage by viewModel.errorMessage.collectAsState()
```

This ensures automatic recomposition when state changes.

## Loading and Error States

Screens handle three states:

1. **Loading** - Show progress indicator
2. **Error** - Display error message with retry button
3. **Success** - Render content

Example from StoryListScreen:

```kotlin
when {
    isLoading -> {
        Box(contentAlignment = Alignment.Center) {
            CircularProgressIndicator(color = AguaSagrada)
            Text("Cargando historias...")
        }
    }
    errorMessage != null -> {
        Column {
            Text(errorMessage ?: "")
            Button(onClick = { viewModel.retry() }) {
                Text("Reintentar")
            }
        }
    }
    else -> {
        LazyColumn { /* Render stories */ }
    }
}
```

See `ui/story/StoryListScreen.kt:88` for state handling.
